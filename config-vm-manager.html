Prep steps

5) Configuring VM Manager

Download virtio (stable) drivers: https://docs.fedoraproject.org/en-US/quick-docs/creating-windows-virtual-machines-using-virtio-drivers/index.html

Download the latest windows 10 iso from Microsoft's website.

You have to start customizing the Virtual Machine before anything else

you must name the VM win10 because of convinience

Select the Q35 chipset and UEFI (not secure boot) as for Firmware

Choose the max Logical Host CPU Core's Based on the Topology

Choose Ram Memory half of your PC's Total Ram

ADD the iso of windows 10 to your Virtual CD ROM Drive

Choose 'Virtual network 'default': NAT for the Temporary Network Card

Install windows like normal.
